# Pharma AI DataOps Platform

## Problem
Pharma data pipelines deal with fragmented, inconsistent, and regulated data (clinical, lab, adverse events). Manual triage and validation slow down operations and introduce risk.

## Solution
Built an end-to-end data platform that:
- Automates ingestion of structured and unstructured pharma data
- Validates incoming datasets with rule-based checks
- Transforms data into curated analytics-ready models
- Uses LLMs for pipeline monitoring, issue triage, and operational insights
